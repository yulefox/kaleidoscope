## 特点

* 基于 HTTP 协议进行应用间的通信

* 保持 SDK 内部的低耦合度

* 为 API 追加签名, 提高安全性

### 对时服务

对时的意义在于使服务器和客户端保持时间一致, 减少逻辑出错几率. 由 SDK 服务器提供 NTP 对时服务, 游戏服务器通过 NTP 同 SDK 服务器进行对时, 玩家客户端通过 HTTP 同 SDK 服务器进行对时.
